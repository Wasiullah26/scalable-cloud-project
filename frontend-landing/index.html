Text to Multiple Languages API

Translate text into several languages with one request. Easy integration via REST API.

How it works (4 steps)

1. Send text

Submit

Send the text you want to translate in a POST request (JSON body with text).

2. Choose languages

Optional

Optionally set source_lang and target_languages. Default: English → Spanish, French, German, Italian, Portuguese.

3. Translate

Process

The API returns the original text plus translations for each requested language.

4. Use in your app

Integrate

Use the JSON response in your frontend. CORS enabled for browser apps.

Try it yourself

API Reference

Base URL: https://t3jb8c44xi.execute-api.us-east-1.amazonaws.com. All responses are JSON.

POST /translate

Send JSON body: { "text": "Your text" }. Optional: source_lang, target_languages (array of codes).

Response: { "original_text": "...", "source_lang": "en", "translations": { "es": "...", "fr": "...", ... } }

GET /health

Check if the API is up.

Response: { "status": "ok", "service": "text-to-languages-api" }

GET /languages

Get default source and target language codes.
